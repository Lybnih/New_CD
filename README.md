# 📦 Case Magalu: Escolha de Novo Centro de Distribuição no Nordeste

### **Análise Estratégica com Dados**

Olá! Neste projeto eu desenvolvi uma **análise de dados estratégica** para ajudar na decisão de onde abrir um novo **Centro de Distribuição (CD)** no Nordeste para o Magazine Luiza.

O estudo compara duas cidades candidatas:

- Recife
- Salvador

A análise foi construída utilizando **Python no Google Colab**, combinando dados públicos, comparações estatísticas e visualizações para chegar a uma recomendação baseada em dados.



# 🎯 Problema

Para melhorar o tempo de entrega e ampliar a eficiência logística no Nordeste, o Magalu precisa definir **a melhor cidade para instalar um novo Centro de Distribuição**.

A dúvida da diretoria está entre duas opções estratégicas:

- Recife  
- Salvador  

Cada cidade possui vantagens diferentes em relação a:

- mercado consumidor
- custo imobiliário
- logística regional
- densidade populacional

A decisão precisa ser **baseada em dados concretos**.


# 📊 Solução

Para resolver esse problema, foi desenvolvido um estudo utilizando **análise exploratória de dados** e **indicadores logísticos**.

A análise utiliza dados públicos do **IBGE** e simulações de mercado imobiliário.

Os principais passos foram:

- 📥 **Coleta de dados demográficos**
- 🔎 **Filtragem de indicadores relevantes**
- ⚖️ **Criação de um sistema de pontuação**
- 📉 **Visualização de dados com gráficos**
- 🏢 **Simulação de custos imobiliários**
- 🚚 **Comparação logística entre capitais do Nordeste**


# 📚 Indicadores analisados

Os principais indicadores utilizados foram:

- **População estimada**  
  Quanto maior a população, maior o potencial de consumo.

- **Densidade demográfica**  
  Populações mais concentradas facilitam a logística de entrega.

- **Área territorial**  
  Impacta na distribuição urbana.

- **Custos imobiliários**  
  Avaliação de preços por metro quadrado para galpões logísticos.


# 📈 Análises realizadas

Durante o estudo foram realizadas diversas análises:

### 📊 Comparação demográfica
Análise de indicadores do IBGE entre Recife e Salvador.

### ⚖️ Score ponderado
Criação de um **sistema de pesos** para calcular qual cidade apresenta melhores condições estratégicas.

### 🏢 Mercado imobiliário
Simulação com anúncios de galpões logísticos para avaliar:

- área média
- preço total
- custo por metro quadrado

### 🚚 Logística regional
Comparação de tempo de entrega para capitais próximas como:

- João Pessoa
- Maceió
- Fortaleza
- Aracaju


# 🧠 Principais Insights

A análise mostrou alguns pontos importantes:

**Salvador**
- Maior população
- Maior potencial de mercado consumidor
- Galpões maiores e mais baratos por m²

**Recife**
- Logística mais rápida para parte do Nordeste
- Localização estratégica para algumas rotas

No balanço geral entre **demanda, custo e logística**, o estudo aponta **Salvador como a opção com melhor custo-benefício**.
